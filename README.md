## Course Introduction

As a second course on programming, the emphasis is that students should be able to write programs of reasonable size and complexity. Devising solutions to problems is encouraged, and converting a design into a computer program is stressed, including software reuse.

The primary focus of the course is to introduce students to object-oriented programming skills. The course provides in-depth coverage of object-oriented programming principles and techniques using **C++20**. Topics include classes, overloading, data abstraction, information hiding, encapsulation, inheritance, polymorphism, file processing, templates, exceptions, container classes, low-level language features, and GUI programming.
